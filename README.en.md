## ğŸ“¦ Installation & Usage

### Chrome / Edge / Other Chromium Browsers

1. **Download Extension**
   ```bash
   git clone https://github.com/0xddy/rplay-downloader.git
   cd rplay-downloader
   ```

2. **Load Extension**
   - Open your browser's extension management page
     - Chrome: `chrome://extensions/`
     - Edge: `edge://extensions/`
   - Enable **Developer mode** in the top right corner
   - Click **Load unpacked**
   - Select the project folder

   **Note: Recent browser versions may require manually enabling "Allow access to file URLs" in the extension details**

3. **Start Using**
   - Visit [rplay.live](https://rplay.live)
   - Play any video
   - Click the extension icon in your browser toolbar
   - Select quality and download
