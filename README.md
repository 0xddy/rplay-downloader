\# 🎬 RPlay Video Downloader



一键下载 rplay.live 视频的浏览器扩展，支持多清晰度选择和实时进度跟踪。



\## ✨ 功能特性



\- 🔍 自动检测 rplay.live 视频流

\- 📊 支持多种清晰度选择（FHD/HD/SD）

\- ⚡ 实时下载进度和速度显示

\- 🔐 自动处理视频解密

\- 🎨 现代化界面设计

\- 🌍 多语言支持



\## 📦 安装使用



\### Chrome / Edge / 其他 Chromium 浏览器



1\. \*\*下载扩展\*\*

&nbsp;  ```bash

&nbsp;  git clone https://github.com/0xddy/rplay-downloader.git

&nbsp;  cd rplay-downloader

&nbsp;  ```



2\. \*\*加载扩展\*\*

&nbsp;  - 打开浏览器扩展管理页面

&nbsp;    - Chrome: `chrome://extensions/`

&nbsp;    - Edge: `edge://extensions/`

&nbsp;  - 开启右上角的 \*\*开发者模式\*\*

&nbsp;  - 点击 \*\*加载已解压的扩展程序\*\*

&nbsp;  - 选择项目文件夹



3\. \*\*开始使用\*\*

&nbsp;  - 访问 \[rplay.live](https://rplay.live)

&nbsp;  - 播放任意视频

&nbsp;  - 点击浏览器工具栏中的扩展图标

&nbsp;  - 选择清晰度并下载
